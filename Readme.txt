This the list of Libraries that should be install before running the code.

pip install transformers datasets torch torchvision torchaudio rouge-score sacrebleu

The data set will be load from datasets library.

https://github.com/EdinburghNLP/XSum/tree/master/XSum-Dataset

BART-T5.ipyn is the code for evaluating BART and T5 models on XSum dataset.
Final.ipyn is the model for with reinforcement learning to fine tune the BART.
